# Changelog

All notable changes to YamanOS are documented in this file.

## v1.0.0 - 2026-02-11

### Added
- Full Tarneeb (Syrian 41) game implementation with reducer/state/AI architecture.
- Trix ruleset profile support (`Classic` and `Jawaker 2025`).
- Generalized doubling flow in Trix (King and profile-based Queens doubling).

### Improved
- Trix AI quality (difficulty behavior and trick-history awareness).
- Trix landscape/portrait responsiveness with safer table/hand layout behavior.
- Mobile viewport handling robustness (`visualViewport` resize/scroll integration).
- Release docs and validation guidance in `README.md`.

### Stability
- Repository-wide JavaScript syntax validation passed.
- Relative import path integrity checks passed.
- CSS/asset reference checks passed.
